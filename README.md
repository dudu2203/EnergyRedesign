# מערכת סקרי אנרגיה ⚡ — v2.8

## תיאור הפרויקט
מערכת ניהול סקרי אנרגיה מלאה בעברית (RTL) לביצוע ותיעוד ביקורות אנרגיה בשטח.  
כל הקוד (HTML, CSS, JavaScript) מוכל בקובץ `index.html` יחיד.

---

## ✅ תכונות שהושלמו (נכון ל-v2.6)

### 🔐 אימות ומשתמשים
- כניסה עם שם משתמש וסיסמה
- תמיכה ב-iOS / Safari (תיקון temporal dead zone)
- שלוש רמות: מנהל מערכת, סוקר
- שמירת סשן ב-localStorage
- ניהול משתמשים מלא (אדמין בלבד): הוספה, עריכה, מחיקה, ייצוא CSV
- שמירת משתמשים ב-API (`app_users`) + localStorage כ-fallback

### 📋 ניהול סקרים (6 שלבים)
1. **פרטי אתר** — שם, כתובת, שטח, שעות, ימי עבודה, פעילות מיוחדת (שישי/שבת/חגים/7-24)
2. **מקורות אנרגיה** — חשמל, גז, סולר, מים ועוד + שנאים + גרפי עונות + ייבוא אקסל  
3. **מערכות ציוד** — תאורה, מיזוג, **מערכות קירור (חדש)**, צ'ילרים, מדחסים, משאבות, מנועים + שורת צרכנים להעמקת סקר  
4. **חללים** — תאורה, מיזוג, מחשבים, מכשירים נלווים  
5. **מסקנות ודליפות אוויר** (חדש) — מסקנות כלליות (מלל חופשי) + תיעוד דליפות אוויר עם תמונה, מיקום, סוג ליקוי, קוטר חור  
6. **סיכום ואישור**
- שמירה אוטומטית (autosave) + שמירה כטיוטה
- **איתור GPS אוטומטי** ממכשיר המשתמש
- סנכרון עם שרת + מצב אופליין
- חיפוש וסינון מתקדם

### 🔄 סינכרון מרובה-משתמשים (חדש ב-v2.6)
- **Polling אוטומטי** — המערכת מושכת עדכונים מהשרת כל 30 שניות ברקע
- **סינכרון ידני** — לחיצה על אייקון הענן מסנכרנת מיידית (push + pull)
- **מחוון מצב** — הצגת סטטוס: 🟢 מסונכרן / 🟠 ממתין לסנכרון / 🔵 מסנכרן... / 🔴 אופליין
- שינויים של משתמשים אחרים מופיעים אוטומטית ברשימה עד 30 שניות
- `syncToServer` גם שולח `systems` בפורמט תקין (תיקון)

### 🛡️ אמינות נתונים (תוקן ב-v2.5)
- רשימת מחוקים (`energy_surveys_deleted`) — סקר שנמחק לא חוזר לעולם
- `mergeSurveys` לפי `updated_at` — מנצח הגרסה החדשה יותר
- retry אוטומטי למחיקה בשרת אחרי חזרה מאופליין
- `saveDraft` לא מחיה סקר שנמחק

### 📤 ייצוא נתונים
- **PDF** — פרטי אתר בדף ראשון בלבד, שאר הדפים ללא כפילות
- **Word (.doc)** — מסמך מלא עם מבנה דפים זהה ל-PDF
- **Excel (.xlsx)** — 4 גיליונות: פרטי אתר / אנרגיה / מערכות / שנאים (SheetJS)
- **CSV** — כל הסקרים עם BOM עברי ל-Excel
- **JSON** — נתוני סקר גולמיים

### 📱 מובייל ו-UX
- עיצוב רספונסיבי מלא
- Service Worker: network-first לעדכונים + תמיכת אופליין
- ניקוי cache אוטומטי בעדכון גרסה
